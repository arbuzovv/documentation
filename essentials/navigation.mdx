Навигация сайта задаётся в файле docs.json. Страница не появляется в боковом меню автоматически: её путь нужно добавить в подходящую группу.

## Пример

~~~json
{
  "navigation": {
    "tabs": [
      {
        "tab": "Руководство",
        "groups": [
          {
            "group": "Начало работы",
            "pages": ["guide/getting-started/overview"]
          }
        ]
      }
    ]
  }
}
~~~

Расширение .mdx в пути не указывается.

## Папки

Путь guide/getting-started/overview соответствует файлу guide/getting-started/overview.mdx. Не используйте api как имя папки верхнего уровня: оно зарезервировано внутренними маршрутами Next.js. Для API-материалов подходит api-reference.

## Скрытые страницы

MDX-файлы, которых нет в docs.json, не отображаются в боковом меню, но могут открываться по прямой ссылке и попадать в поиск.

<Warning>
  После изменения навигации выполните mint validate и mint broken-links.
</Warning>
