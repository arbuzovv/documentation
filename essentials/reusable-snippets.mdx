## Создание фрагмента

Поместите MDX-файл в каталог snippets. Файлы из этого каталога считаются компонентами и не публикуются как самостоятельные страницы.

~~~mdx
Фактическая доступность инструмента зависит от типа счёта и прав пользователя.
~~~

Импортируйте фрагмент в нужную страницу:

~~~mdx
import AccessNote from '/snippets/access-note.mdx';

<AccessNote />
~~~

## Переменные

Фрагмент может экспортировать значения:

~~~mdx
export const marketName = 'Московская биржа';
~~~

Их можно импортировать в другую страницу:

~~~mdx
import { marketName } from '/snippets/market.mdx';

Основной рынок в примерах — {marketName}.
~~~

## Компоненты

Для повторяющейся разметки можно экспортировать React-компонент:

~~~mdx
export const ProductLabel = ({ name }) => (
  <strong>{name} от Финама</strong>
);
~~~

<Warning>
  В теле стрелочной функции используйте JSX/HTML. Обычная MDX-разметка там не компилируется.
</Warning>
