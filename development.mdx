<Info>
  Для локальной работы нужен **Node.js 20.17.0 или новее**. Рекомендуется актуальная LTS-версия.
</Info>

## Установка Mintlify CLI

~~~bash
npm i -g mint
~~~

Перейдите в корень репозитория, где находится docs.json, и запустите:

~~~bash
mint dev
~~~

Предпросмотр откроется по адресу http://localhost:3000.

### Другой порт

~~~bash
mint dev --port 3333
~~~

Если выбранный порт занят, CLI предложит следующий свободный.

## Обновление CLI

~~~bash
mint update
~~~

Если команда недоступна в установленной версии:

~~~bash
npm i -g mint@latest
~~~

## Проверки перед публикацией

~~~bash
mint validate
mint broken-links
~~~

Также выполните поиск по словам Limex, Lime Trading, AAPL, NYSE, NASDAQ, USD и US. Такие термины допустимы только в явно обозначенном историческом или миграционном контексте.

## Редактирование MDX

Для подсветки и форматирования удобно использовать расширения MDX и Prettier для VS Code, Cursor или другого совместимого редактора. Не переводите имена функций, аргументов, полей API и Python-идентификаторы.

## Устранение неполадок

<AccordionGroup>
  <Accordion title="Не загружается модуль sharp на macOS с Apple Silicon">
    Удалите старую CLI командой npm uninstall -g mint, обновите Node.js до версии 20.17.0 или новее и снова выполните npm install -g mint.
  </Accordion>

  <Accordion title="CLI завершает работу с неизвестной ошибкой">
    Удалите локальный кеш ~/.mintlify, обновите CLI и повторно запустите mint dev.
  </Accordion>

  <Accordion title="Локальная версия отличается от опубликованной">
    Выполните mint update и перезапустите локальный сервер.
  </Accordion>
</AccordionGroup>
