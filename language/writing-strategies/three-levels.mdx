---
title: "Три уровня разработки стратегии"
description: "Чат, готовые блоки или полный код на Python"
---

Чтобы начать работу, необязательно сразу писать код. **Ziplime от Финама** поддерживает три способа создания стратегии, между которыми можно свободно переключаться.

<Tabs>
  <Tab title="Чат">
    Опишите идею обычным языком, и [команда ИИ-агентов](/guide/platform/ai-agent-team) подготовит код на Python: подключит данные, исправит ошибки и предложит варианты оптимизации. Знание программирования не требуется.
  </Tab>
  <Tab title="Готовые блоки">
    Добавляйте в сгенерированный код готовые блоки — индикаторы, источники данных или правила выставления заявок. Окружающий код будет обновлён с учётом выбранного блока. Так можно использовать дополнительные возможности, не создавая стратегию на Python с нуля.
  </Tab>
  <Tab title="Полный код">
    Пишите и редактируйте код стратегии на Python напрямую через API Ziplime/Zipline, без участия ИИ. Этот уровень подходит, когда нужен полный контроль: собственная обработка данных, нестандартная логика заявок или код, перенесённый с другой платформы.
  </Tab>
</Tabs>

На каждом уровне результатом становится стратегия на Python для одного и того же движка — независимо от того, написан код вручную или сгенерирован по описанию в чате. Для реального выставления заявок через брокерский контур Финама используется [Finam Trade API](https://api.finam.ru/).
