---
title: "Three Levels of Working with the Platform"
description: "A platform that grows with you"
---

Ziplime is designed for gradual growth — from complete beginner to experienced developer. The platform scales with the user rather than forcing everyone through the same workflow.

<Steps>
  <Step title="Level 1 — Chat">
    Just write in the chat and talk to the agent team. This is the lowest possible barrier to entry — no special knowledge is required.
  </Step>
  <Step title="Level 2 — Code blocks">
    Add ready-made blocks directly into the code, and the code is rewritten to incorporate them. This unlocks hundreds of indicators and data sources that are hard to reach through chat alone.
  </Step>
  <Step title="Level 3 — Full code">
    Read and edit the code directly, without going through the AI. Maximum control and flexibility.
  </Step>
</Steps>

One product covers the needs of both an absolute beginner and an experienced quantitative analyst.
