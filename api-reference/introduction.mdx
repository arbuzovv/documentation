Библиотека Ziplime позволяет разрабатывать и запускать стратегии локально. Она предназначена для пользователей, которым нужен полный контроль над Python-кодом, источниками данных и инфраструктурой выполнения.

## Требования

- Python 3.12 или новее;
- отдельное виртуальное окружение;
- источник исторических данных с российскими инструментами;
- корректный календарь Московской биржи;
- для реальной торговли — брокерский счёт и доступ к Финам Trade API.

## Установка

~~~bash
python -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install ziplime
~~~

В Windows активируйте окружение командой .venv\Scripts\activate.

## Структура стратегии

~~~python
from ziplime.finance.execution import MarketOrder


async def initialize(context):
    context.asset = await context.symbol("SBER")
    context.invested = False


async def handle_data(context, data):
    if not context.invested:
        await context.order_target_percent(
            asset=context.asset,
            target=1.0,
            style=MarketOrder(),
        )
        context.invested = True
~~~

Перед запуском зарегистрируйте набор данных, в котором тикер SBER однозначно связан с нужной площадкой и режимом торгов. Для календаря Московской биржи в совместимых версиях exchange_calendars используется код XMOS.

## Локальный бэктест

CLI и параметры запуска меняются вместе с библиотекой. Проверьте команды установленной версии:

~~~bash
python -m ziplime --help
python -m ziplime run --help
~~~

Не переносите старые команды с наборами данных Limex или Lime Trading: они относятся к другому контуру.

## Реальная торговля через Финам

Для брокерского подключения используйте [Финам Trade API](https://api.finam.ru/). Доступность инструментов определяется типом счёта, пройденными тестами, статусом инвестора и текущими условиями брокера.

<Steps>
  <Step title="Откройте или подключите счёт">
    Используйте действующий брокерский счёт Финама или демо-контур, если он доступен для выбранного сценария.
  </Step>

  <Step title="Создайте токен">
    Получите секрет в личном кабинете Trade API и обменяйте его на JWT по официальной схеме авторизации.
  </Step>

  <Step title="Настройте адаптер">
    Убедитесь, что установленная версия интеграции Ziplime поддерживает нужные методы Финам Trade API, площадку и тип заявки.
  </Step>

  <Step title="Проверьте риск">
    Начните с демо-режима или минимального объёма, задайте лимиты позиции и обработку ошибок соединения.
  </Step>
</Steps>

<Warning>
  Не храните секреты и JWT в репозитории, алгоритмическом файле или документации. Используйте переменные окружения либо защищённое хранилище секретов.
</Warning>

<Note>
  Интерфейсы библиотек и брокерского API развиваются независимо. Перед реальной торговлей сверяйте версию Ziplime с актуальной документацией Финам Trade API.
</Note>
